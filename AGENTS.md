# Story

A simple creative writing repository containing two short text chapters. This is not a software project — it is a personal text/story repository with no code, build system, or tooling.

## Contents

- `chapter1.txt` — First chapter of a short story
- `chapter2.txt` — Second chapter of a short story

## Conventions & Notes for Agents

- This repository contains plain text files only — no code, no build system.
- There is nothing to install, build, or test.
- If adding new chapters, follow the naming pattern `chapter<N>.txt`.
